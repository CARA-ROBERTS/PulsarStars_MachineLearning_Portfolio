What is a Random Forest?

A random forest is a meta estimator that fits numerous decision tree classifiers on various sub-samples of the dataset and uses averaging to improve the predictive accuracy and control over-fitting.

https://scikit-learn.org/stable/modules/generated/sklearn.ensemble.RandomForestClassifier.html

Creating a Random Forest for Predict Model

The predicted outcome (y_pred) of the model can then be compared to the actual outcome (y_test) of Pulsar Stars. The accuracy of model can be scored by the comparison. The Random Forest model scored an excellent accuracy of approximately 98% on the test data. The accuracy should be as close to 100% without reaching 100%. Any perfect score reflects an overfitted model. An overfitted model can have a lower accuracy for a new dataset prediction, thus an ineffective model.

Initial set up for Random Forest prediction model starts with setting the "x" variable with all columns (classifiers) and data without the target predict column (Pulsar Stars). The "y" variable consists only the target predict column (Pulsar Star). With the "X" and "y" set, the train_test_split can be conducted across the dataset. A 75% train data and 25% test is utilized for the model. The Random Forest classifiers are set to 100 estimators, then fit to the train data. The same Random Forest classifiers are then called to predict on test data.

Confusion Matrix to Analyze Predict Results

The confusion matrix is an indepth method to analyze the Pulsar Stars predicted result. Retrieving the statistics of the confusion matrix, TP(true positive), FP(false positive), TN(true negative), and FN (false negative) are important method to further analyze the predict results. Higher TP (true positives) and TN(true negatives) than FP (false positives) and FN (false negative) showcase a more acurrate model. FP and FN should be as low as possible for optimal results for predictions. The Seaborn generated Confusion Matrix Heatmap visualizes the TP, FP, TN, and FN with the parameters of 0 (not Pulsar), 1 (Pulsar) and all results.
