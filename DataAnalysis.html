Original Star Data Analysis

Analysis of Data Before Trained and Modeled.

Aggregation and visualization highlight any outlier and bias perspective when analyzing data. Once understanding the original dataset, data munging and preprocessing are set to prepare data for machine learning models. The dataset has eight variable Classifier columns and one predict Target column. The Classifier columns will weight the significance of the data for the prediction. Classifier columns must be revelant toward the predict Target column result. Any Classifer column not relevant will dropped during data munging. The one predict Target column in this data set is target_class column. The target_class column in this dataset is the Puslar Star. The target_class column has data of binary 0 or 1 result. Value of 0 equates to No, not a Pulsar Star. Value of 1 equates to Yes, Pulsar Star..

Aggregation of the Dataset

The Original Star Data contains 9 columns and 17898 rows of data, which is sufficient amount to have an accurate projection of analysis, trained data, and prediction model. Each row is a star observation.

The higher count of the star observations, the more increases the training ability for better accuracy of the model predictions of Pulsar Stars. More data is beneficial for modeling as long as it is relevant for the target prediction.
