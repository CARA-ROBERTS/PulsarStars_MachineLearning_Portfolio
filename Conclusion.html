Pulsars Stars Machine Learning Comparison

The Random Forest and Neural Net models have a close accuracy rate of approximately RF(0.9798) and NN(0.9785). Both rates are efficiently high for Pulsar Star predictions. Each model has its own attibutes for making predictions.

Random Forest models showcases the feature importance of the classifiers in the prediction model sequence. The vital information on the weighted classifier can determined important column to focus on for further analysis toward achieving optimal target results. Also, the Random Forest function Confusion Matrix adds pertinent infomation to the predictive results of the Random Forest model.

Neural Network models can be adjusted to specific method of creation of a model by layers, how the model is compiled/ fit, and the manner the model is trained for optimal results applicable for dataset and its target result. Neural Net models are more dynamic in the abililty to adjust the input, hidden, output layers with number of hidden nodes and type of activation. Also, the model can be adjusted how it is compiled based on how its optimized, loss, and metrics style. Further depths on model fitting specifications tell the created model how to train on the given dataset with shuffle and ammount of epochs. Each methodically step in the building model process can be adjusted for optimal results.

Among the various machine learning techniques, Random Forest and Neural Network return the best results for a binary target/predict result. The utilization of both methods is best for comparison dataset and predictive results.
